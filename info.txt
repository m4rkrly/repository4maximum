Виртуальное окружение - это папка для хранения библиотек  

Чтобы создать виртуальное окружение, мы должны в терминале прописать это:
"python -m venv venv" 

Далее необходимо активировать виртуальное окружение. Для этого необходимо запустить файл "activate"
вот таким образом: "venv/Scripts/activate"

Для создания проекта (а соответственно и сайта) нам необходимо сначала прописать команду:

django-admin startproject названиепроекта . (точку по желанию, она предотвращает создание дополнительной
папки под проект)

Затем стоит запустить его командой:

python manage.py runserver

web-application(app) - это отдельная надстройка в виде логической части, которая отвечает за определённый
функицонал сайта(логин/логаут, платежи, профиль, главные страницы)

Для создания веб-приложения нам нужна команда:

python manage.py startapp название

После этого в корневой папке в файле "settings.py" необходимо в классе "INSTALLED_APPS" указать новое
приложение (просто написать его название)
